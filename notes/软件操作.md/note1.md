### 版本库操作
#### svn
```
新增文件 svn add
提交 svn commit -m"xx"
更新 svn up
查看状态 svn st
```
#### git
```
新增： git add.
提交：git commit -m"xxx"
更新：git pull
放弃本地修改：git reset --hard
更新到主分支：git push origin master
```
